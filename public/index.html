TAQL Bookmarklets

TAQL can be enabled/disabled explicitly in TA minisites via the 'GraphQLNextGen' cookie. The following bookmarklets can be used to set the cookie while visiting hare/lapin. To install the bookmarklets, drag the links to your browser's bookmarks bar. When viewing webml.dev.tripadvisor.com or hare.tripadvisor.com, click the appropriate bookmark(let) to force taql on or off, or to force the default behavior.

  • TAQL: Disable explicitly
  • TAQL: Use environment default
